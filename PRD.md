## 1. Проблема

Большинство детских приложений в App Store:
- Перегружены рекламой и встроенными покупками
- Слишком сложны для детей младше 3 лет (мелкие кнопки, сложные жесты)
- Используют синтетические голоса вместо естественных
- Не учитывают особенности моторики малышей (неточные касания, случайные нажатия)
- Контент на английском, русскоязычных качественных приложений мало

## 2. Решение

Нативное iOS-приложение с крупными яркими элементами, реальными звуками и простым управлением (только тап). Полностью на русском языке. Без рекламы, без интернета, без покупок.

### Принципы:
- **Один тап — один результат.** Никаких свайпов, долгих нажатий, drag-and-drop
- **Крупные элементы.** Минимальная зона касания — 80pt (стандарт Apple — 44pt, для малышей удваиваем)
- **Мгновенная обратная связь.** Каждое касание — звук + анимация за < 100мс
- **Короткие сессии.** Каждая активность — 1-3 минуты
- **Позитивное подкрепление.** Нет понятия "ошибка" — только поощрение
- **Безопасность.** Нет рекламы, нет ссылок, нет интернета, родительский замок

---

## 3. Целевая аудитория

**Основная:** дети 1.5 — 3 года
**Вторичная:** родители, которые ищут полезный контент для ребёнка

### Навыки развития:
- Распознавание звуков (животные, транспорт, природа, музыкальные инструменты)
- Связь "картинка → звук → слово"
- Изучение цветов
- Изучение базовых форм
- Развитие мелкой моторики (точное касание)
- Причинно-следственные связи (нажал → получил результат)
- Счёт от 1 до 5

## 5. Игровые режимы

### 5.1. Звуки животных

**Экран:** сетка 2x2 (или 2x3 на iPad) с крупными карточками животных.

**Механика:**
1. Ребёнок видит 4 карточки с иллюстрациями животных
2. Тапает на карточку
3. Карточка увеличивается с пружинной анимацией
4. Воспроизводится реальный звук животного (не синтезированный)
5. Через 0.5 сек — голос произносит название: "Это кошка. Мяу!"
6. Карточка возвращается на место
7. Появляются звёздочки / конфетти

**Контент (MVP):**
- Кошка, собака, корова, лошадь
- Курица, утка, свинья, овца
- Лев, слон, обезьяна, волк
- Лягушка, птица, сова, змея

**Навигация:** стрелки влево/вправо для переключения между наборами по 4 карточки.

| ID | Требование | Приоритет |
|---|---|---|
| AN-001 | Сетка 2x2 карточек с иллюстрациями | P0 |
| AN-002 | Тап → звук животного + название голосом | P0 |
| AN-003 | Пружинная анимация при тапе | P0 |
| AN-004 | Минимум 16 животных | P0 |
| AN-005 | Листание наборов стрелками | P0 |
| AN-006 | Случайный порядок карточек при каждом входе | P1 |

**Контент (MVP):**

- Телефон, часы, дверной звонок, барабан
- Дождь, гром, ветер, море
- Пожарная машина, скорая помощь, полиция, велосипед

### 5.3. Угадай звук

**Экран:** вверху — кнопка "Послушай" с динамиком, внизу — 2 карточки (варианта).

**Механика:**
1. Автоматически воспроизводится звук
2. Ребёнок выбирает из 2 карточек (не 4 — для 2-летнего 2 варианта достаточно)
3. Правильный ответ → звёздочки + "Молодец!" + переход к следующему
4. Неправильный → карточка мягко покачивается, звук повторяется, правильная карточка подсвечивается

| ID | Требование | Приоритет |
|---|---|---|

| GU-002 | Кнопка повтора звука | P0 |
| GU-003 | При неправильном — подсказка, не наказание | P0 |
| GU-004 | Автопереход к следующему после правильного | P0 |
| GU-005 | Звуки берутся из наборов 5.1 и 5.2 | P0 |

### 5.4. Цвета

**Экран:** большой круг/квадрат определённого цвета + голос называет цвет.

**Механика:**
1. На экране крупная фигура одного цвета
2. Голос произносит: "Это красный цвет!"
3. Тап на фигуру — она превращается в предмет этого цвета (красный → яблоко, клубника)
4. Свайп или тап стрелки → следующий цвет

**Контент (MVP):** красный, синий, жёлтый, зелёный, оранжевый, фиолетовый, белый, чёрный

| ID | Требование | Приоритет |

| CL-004 | Игра "Найди такой же цвет" (2 варианта) | P1 |

### 5.5. Формы

**Экран:** крупная фигура + голос.

**Механика:**
1. На экране — форма (круг, квадрат, треугольник, звезда)
2. Голос: "Это круг!"
3. Тап — форма вращается/подпрыгивает с весёлой анимацией
4. Стрелки → следующая форма

**Контент (MVP):** круг, квадрат, треугольник, звезда, сердце, овал

### 5.6. Считаем

**Экран:** предметы появляются по одному, голос считает.

**Механика:**
1. Экран пуст
2. Тап → появляется первый предмет (яблоко), голос: "Один!"
3. Тап → второй предмет, голос: "Два!"
4. До 5 предметов
5. После 5 — конфетти + "Молодец! Ты посчитал до пяти!"

### 5.7. Музыкальная панель

**Экран:** 4-6 крупных кнопок-инструментов.

**Механика:**
1. На экране — барабан, бубен, ксилофон, маракасы, колокольчик, пианино
2. Тап → звук инструмента + вибрация + визуальная волна
3. Ребёнок свободно тапает, создаёт свои "мелодии"
4. Нет правильного/неправильного — полная свобода

## 6. Главный экран

**Дизайн:** яркий фон (небо + облака / поляна), 4-6 крупных круглых иконок режимов. Каждая иконка — понятная картинка (кошка для звуков, палитра для цветов, барабан для музыки).

**Навигация:**
- Тап на иконку → переход в режим
- Кнопка "Назад" (домик) в каждом режиме — в левом верхнем углу, крупная
- Родительский замок — иконка шестерёнки с защитой от ребёнка (нужно зажать 3 сек)

## 9. Звуковой контент

### Требования к звукам:
- **Животные:** реальные записи (не синтезированные)
- **Транспорт:** реальные записи
- **Голос:** профессиональная озвучка, чёткая дикция, дружелюбный тон
- **Формат:** AAC (.m4a), 44.1 kHz
- **Длительность:** звук — 1-2 сек, слово — 1-3 сек

### Источники (бесплатные):
- freesound.org (Creative Commons)
- pixabay.com/sound-effects
- Собственная запись голоса для слов

### Структура файлов:
```
Resources/Sounds/
├── animals/
│   ├── cat_sound.m4a          # мяу
│   ├── cat_word.m4a           # "Это кошка!"
│   ├── dog_sound.m4a
│   └── ...
├── transport/
│   ├── car_sound.m4a
│   ├── car_word.m4a
│   └── ...
├── instruments/
│   ├── drum.m4a
│   └── ...
├── ui/
│   ├── correct.m4a            # правильный ответ

│   ├── try_again.m4a          # "Попробуй ещё раз!"

    ├── one.m4a
    ├── two.m4a
    └── ...
```

## 10. Визуальный стиль

### Палитра:
- Фон: мягкий голубой (#E3F2FD) или тёплый жёлтый (#FFF9C4)
- Акценты: яркие чистые цвета (красный, зелёный, оранжевый, фиолетовый)
- Карточки: белые с тенью и скруглением 20pt
- Без градиентов — плоские чистые цвета

### Иллюстрации:
- Стиль: flat, дружелюбный, с крупными чертами
- Животные: мультяшные но узнаваемые (не абстрактные)
- Минимум деталей — ребёнок должен мгновенно узнать
- SVG формат для масштабирования

### Анимации:
- Тап: scale 1.0 → 1.15 → 1.0 (пружина, 0.3 сек)
- Правильный ответ: конфетти из звёздочек (1 сек)
- Переход между экранами: slide, 0.25 сек
- Появление карточек: fade in + slide up, 0.2 сек с задержкой

## 11. Модель данных

### GameItem
```
- id: UUID
- category: Category (animal, transport, nature, instrument)
- name: String              // "cat"
- displayName: String       // "Кошка"
- soundFile: String         // "cat_sound.m4a"
- wordFile: String          // "cat_word.m4a"
- imageName: String         // "cat" (asset name)
- color: Color?             // для режима цветов
- shape: Shape?             // для режима форм
```

### UserProgress
```
- totalSessions: Int
- totalTime: TimeInterval
- modeUsage: [GameMode: Int]      // сколько раз открывал режим
- lastPlayed: Date
```

### Settings
```
- backgroundMusic: Bool
- hapticEnabled: Bool
- voiceType: VoiceType (male, female)
- timeLimit: Int?           // минуты, nil = без ограничения
```

## 15. App Store

### Категория: Kids (Ages 5 & Under)

### Требования Apple для детских приложений:
- Нет рекламы
- Нет сторонних SDK аналитики
- Нет ссылок на внешние сайты
- Privacy Policy обязательна
- Parental gate для любых покупок/настроек

### Описание:
> TinyMind — развивающая игра для самых маленьких. Учим звуки животных, цвета, формы и счёт. Простое управление одним пальцем, яркие иллюстрации и живые звуки. Полностью на русском языке. Без рекламы, без интернета.
